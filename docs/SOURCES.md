# Sources Policy

External sources are reviewed references until approved.

## Intake Rules

- Record the source before extracting from it.
- Prefer official and maintained sources.
- Check license and trust level.
- Check safety before usefulness.
- Do not install, activate, clone, or run external content during initial evaluation.

## Source States

- Candidate.
- Under review.
- Approved for method extraction.
- Reference only.
- Ignored.
- Quarantined.
- Install later after approval.
