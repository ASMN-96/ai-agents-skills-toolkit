# Registry Frontmatter Template

Use this as a starting point for future source files that may become registry-generation inputs.

```yaml
---
name:
description:
registryId:
registryType:
sourceRef: ["unknown-review-required"]
lastExtracted: unknown-review-required
status: draft
---
```

Do not use frontmatter to grant trust, license approval, security approval, runtime activation, routing authority, tool permissions, or public release readiness.
