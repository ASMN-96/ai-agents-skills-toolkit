# Templates

Templates support consistent project context, release gates, state tracking, source/tool records, registry frontmatter, compiled-agent metadata, Level 4 evidence, and decision logs.

They are starting points for future compiled-agent and project-sync workflows.

Level 4 templates:

- `level4-project-pilot-report.template.md`
- `level4-rollback-rehearsal-report.template.md`
- `level4-enterprise-tool-review.template.md`
- `level4-warning-threshold-decision.template.md`

Governance templates:

- `governance/stop-condition-message-templates.md`
