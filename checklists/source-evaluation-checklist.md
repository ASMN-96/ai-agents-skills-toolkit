# Source Evaluation Checklist

Use this checklist for external skills, repositories, marketplaces, official docs, and community sources.

## Metadata

- Source name.
- Source URL.
- Source type.
- Owner or publisher.
- License.
- Last update.
- Visible adoption signals.

## Safety

- Prompt-injection risk.
- Dangerous scripts.
- Shell commands.
- Network calls.
- Secret access.
- Filesystem access.
- Conflicting instructions.

## Usefulness

- Relevant methods.
- Applicable agents.
- Required extraction work.
- Known limitations.

## Classification

Choose one:

- Extract into methods.
- Reference only.
- Ignore.
- Install later after approval.
