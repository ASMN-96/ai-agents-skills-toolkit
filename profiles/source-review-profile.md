# Source Review Profile

## Included Agents

- Skill Scout Agent
- Security Agent
- Architect Agent
- Reviewer Agent

## Recommended Support Tools

- GitHub/gh or web search/browser for source identity checks when explicitly needed.
- Superpowers for verification honesty and source-safety discipline.
- GSD Core as active-if-detected governed tool metadata for serious source-adoption and refresh programs when available; owner-approved install/config only when absent.

## Default Mode

Read-only review.

## Allowed Actions

- Evaluate external sources, tools, skills, source records, license, trust, maintenance, and useful patterns.
- Classify sources for method extraction, reference-only use, archive, removal, or rejection.
- Recommend normalized methods, routing, evals, or validators without importing raw upstream content.

## Forbidden Actions

- Install, activate, clone, run unknown scripts, configure MCP, or change global config.
- Copy raw upstream prompts, components, registry JSON, package config, or scripts into active paths.
- Treat source metadata as approval to execute tools or sync product repositories.

## Required Output Format

- Source identity and reviewed evidence.
- Trust/license/security findings.
- Useful normalized value path.
- Rejected operations and owner decisions.

## Required Verification Gates

- Confirm source record and watchlist status when available.
- Confirm dangerous commands, prompt-injection risk, and secret/network access risk were considered.
- Confirm no install, activation, raw copy, or product-repo change occurred.
