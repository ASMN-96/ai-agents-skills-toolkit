# Compiled Agent Metadata Template

```yaml
---
toolkit_name: AI Agent Skills Toolkit
toolkit_version:
toolkit_pin:
compiled_status: approved|review
compiled_at: deterministic-not-recorded
source_commit:
source_agent:
compiler: scripts/compile-agents.mjs
registry_input: registries/agents.registry.json
source_profile_refs: []
source_method_refs: []
compile_contract_version:
---
```

Metadata must be generated by a reviewed deterministic compiler. Approved registry agents compile as `approved` only when the source agent passes source-quality and parity validation; draft or unapproved agents compile as `review`. Do not mechanically restamp compiled agents without regenerated provenance and review evidence.
